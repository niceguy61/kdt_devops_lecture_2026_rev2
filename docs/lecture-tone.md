# Lecture Tone And 운영 준수 기준

이 문서는 계약서 주의사항을 강의 제작과 강의 운영에 반영하기 위한 기준이다. 강의안은 학생에게 신뢰를 주고, 정해진 시간과 주교재 중심 흐름을 지키며, 교육과 무관한 잡담/홍보/불평을 배제해야 한다.

## 소개
| 해야 할 것 | 피해야 할 것 |
|---|---|
| 강의 경험, 현장 경력, 수업 운영 방식을 충분히 소개하여 훈련생에게 신뢰를 준다. | "처음이다", "처음 해본다"처럼 훈련생 신뢰를 낮출 수 있는 표현을 하지 않는다. |

## 강의 운영
| 해야 할 것 | 피해야 할 것 |
|---|---|
| 강의 계획서와 세부 강의 시간을 준수한다. | 장시간의 교육과 관련 없는 잡담을 하지 않는다. |
| 세부 강의 계획을 안내하고 완성도 높은 강의를 위해 사전 준비한다. | 강의 중 핸드폰 사용을 하지 않는다. |
| 전일 강의 내용을 필요한 범위에서 연결하고, 교육 종료 시 당일 교육 내용을 정리한다. | 부교재 중심으로 흐르지 않게 하고 주교재를 기준으로 수업한다. |
| 교육장 상황과 훈련생 특이사항을 확인하고 필요한 경우 통보한다. | 교보재, 교육장 환경, 운영자에 대한 개인 불만을 훈련생에게 말하지 않는다. |
| 난이도에 맞는 시험 문제를 출제한다. | 부적절한 강의 자세를 취하지 않는다. |
| 출석부 강사란에 정자체로 서명하고, 매시간 출결 사항을 확인한다. | 개인 업무, 소속사, 상품 소개/홍보를 하지 않는다. |
| 실습 강의를 위해 네트워크 환경과 마이크 등 소통 환경을 준비한다. | 훈련생과 언쟁하지 않는다. |
| 강의 skill 향상을 위해 지속적으로 개선한다. | 쉬는 시간과 점심 시간에 강의와 무관한 웹 서핑, 만화, 동영상, 게임 등을 하지 않는다. |
|  | 교육과 관계없는 상담, 예를 들어 이직 상담 등으로 수업을 흐리지 않는다. |

## 교육 시간
| 해야 할 것 | 피해야 할 것 |
|---|---|
| 교육 시간을 임의로 조정하지 않는다. 부득이한 경우 회사와 사전 협의 후 진행한다. | 강의 시간에 지각하지 않는다. |
| 50분 수업과 10분 휴식을 기준으로 운영한다. | 강사 임의로 수업 시간을 조정하지 않는다. |
| 교육 시작 10분 전 입실한다. | 점심 시간을 임의로 조정하지 않는다. |
| 점심시간과 교육 종료 시간을 준수한다. |  |

## 품위 유지
| 해야 할 것 | 피해야 할 것 |
|---|---|
| 품위에 맞는 복장과 언행을 유지한다. | 과도한 화장, 향수, 액세서리 또는 강의에 부적절한 복장을 피한다. |
| 훈련생과 과정 운영자에게 경어를 사용한다. | 설문과 강사료의 관계를 언급하지 않는다. |

## 교육 종료 이후
| 해야 할 것 | 피해야 할 것 |
|---|---|
| 수료 테스트 진행에 적극 참여한다. | 훈련생 문의에 대한 feedback을 늦게 처리하지 않는다. |
| 이후 연계 과정 안내를 필요한 범위에서 제공한다. |  |
| 훈련생 문의에 대한 feedback 기준을 사전에 공지하고 약속을 지킨다. |  |

## 강의안 반영 규칙
- 강의안은 학생에게 불안감을 주는 자기비하, 준비 부족, "처음" 표현을 포함하지 않는다.
- 시간표는 50분 수업, 10분 휴식, 점심시간, 종료 시간을 존중하는 형태로 작성한다.
- OT에는 강의 경험/현장 경력/운영 방식 소개 시간이 있어야 한다.
- 수업 본문에는 교육과 무관한 잡담, 개인 홍보, 상품 홍보, 교육장/교보재 불평, 언쟁 유도 표현을 넣지 않는다.
- 학생을 지칭할 때는 경어와 존중 표현을 사용한다.
- 종료 전에는 당일 학습 내용, 산출물, 다음 행동, feedback 경로를 정리한다.
